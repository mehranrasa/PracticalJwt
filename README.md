# PracticalJwt
Real world jwt example with rotating refresh tokens & encrypted jwt payload

# Technologies, Tools, Patterns used :
• Simple Domain Driven Design

• EFCore (Sqlite db)

• JWT Tokens (with payload encyription)

• CQRS Pattern (with MediatR)

• Repository Pattern
